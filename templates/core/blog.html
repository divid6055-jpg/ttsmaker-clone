Notes & updates

From prototype to production

This project started as a static clone of TTSMaker. It now ships with real Microsoft Azure neural voices via edge-tts, a curated honest voice catalog, per-IP daily quotas, rate limiting, real format conversion via ffmpeg, automatic file cleanup, and production security defaults.

See the API docs for the current endpoints.

Arabic and RTL support

The UI flips to right-to-left direction automatically for Arabic, Urdu, Persian and Hebrew. Voice coverage for Arabic includes Egyptian, Saudi, Jordanian, Lebanese, Emirati, Iraqi, Syrian and Tunisian variants — all backed by real neural voices.
