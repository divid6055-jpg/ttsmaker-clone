Why this clone is different

Real neural voices

Every voice ID maps to a real Microsoft Azure neural voice through edge-tts. No fake or mirrored IDs.

Full Arabic & RTL

First-class Arabic support with Egyptian, Saudi, Jordanian, Lebanese, Emirati and more dialects. The UI flips to RTL when you pick a right-to-left language.

Production hardened

Secret keys via env, locked-down CORS, per-IP daily quota, per-minute rate limiting, automatic cleanup of old audio files.

Honest format support

MP3 always works. WAV, OGG, AAC are offered only when ffmpeg is installed — no silent mislabelling.
