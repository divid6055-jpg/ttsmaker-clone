GET Get Voice List

Retrieve a list of all available languages and voices with their IDs, character limits, and supported emotion styles.

GET https://ttsmaker-clone.example.com/api/voices/?language=en

Parameters

ParameterTypeRequiredDescription
languagestringOptionalFilter by language code (e.g., en, ar, zh, es, fr, de)

Response Example

{
  "error_code": 0,
  "msg": "Query the list of voices successfully.",
  "language": "en",
  "support_language_list": ["en", "ar", "zh", ...],
  "voices_count": 26,
  "voices_detailed_list": [
    {
      "id": 148,
      "name": "Alayna-United States Female",
      "language": "en",
      "text_characters_limit": 20000,
      "unlimited": true
    }
  ]
}

Supported Language Codes

en, ar, zh, es, pt, de, fr, it, tr, ru, ja, ko, ms, th, vi, id, he, fi, sv, nb, da, hi, ro, hu, pl, nl, cs, uk, hr, el, sk, bg, sl, ca, ga, lt, fa, sw, bn, ur, ta, mr, te, gu, ml, kn

POST Create TTS Order

Submit a text-to-speech conversion order. Returns a temporary download URL for the generated audio file.

POST https://ttsmaker-clone.example.com/api/tts/

Request Body (JSON)

ParameterTypeRequiredDescription
textstringRequiredText to convert (max 20,000 chars for unlimited voices)
voice_idintegerRequiredVoice ID from the voice list
audio_formatstringRequiredmp3, wav, ogg, aac, opus (default: mp3)
audio_speedfloatOptionalSpeed rate (0.5 - 2.0, default: 1.0)
audio_volumefloatOptionalVolume level (0.1 - 2.0, default: 1.0)
audio_pitchfloatOptionalPitch adjustment (0.5 - 2.0, default: 1.0)
audio_high_qualityintegerOptional0 or 1 for high quality (default: 0)
text_paragraph_pause_timeintegerOptionalPause between paragraphs in ms (0-10000)
emotion_style_keystringOptionalEmotion style for supported voices
emotion_intensityfloatOptionalEmotion intensity (0.1 - 2.0, default: 1.0)

Request Example

curl -X POST https://ttsmaker-clone.example.com/api/tts/ \
  -H "Content-Type: application/json" \
  -d '{
  "text": "Hello, this is a test of text to speech.",
  "voice_id": 148,
  "audio_format": "mp3",
  "audio_speed": 1.0,
  "audio_volume": 1.0,
  "emotion_style_key": "cheerful"
}'

Success Response

{
  "error_code": 0,
  "msg": "TTS Task Executed Successfully",
  "tts_task_characters_count": 45,
  "audio_download_url": "https://example.com/api/download/tts_12345.mp3",
  "audio_file_format": "mp3",
  "audio_file_expiration_timestamp": 1723560282
}

Error Codes

Error CodeSummaryDescription
-1POST_FIELD_ERRORPost data fields are incomplete
-1VOICE_ID_ERRORVoice ID does not exist or is disabled
-1TEXT_LENGTH_ERRORText exceeds maximum character limit
-1ERROR_TTS_FAILEDServer-side conversion failure
0TTS Task Executed Successfully

GET Check Token Status

Retrieve information about character quota usage and account status.

GET https://ttsmaker-clone.example.com/api/token-status/

Response Example

{
  "error_code": 0,

  "account_status": {
    "quota_characters": 20000,
    "characters_used": 500,
    "available_quota": 19500,
    "subscription_period": "week"
  }
}
