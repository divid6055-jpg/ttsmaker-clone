API Documentation

The TTSMaker Clone exposes a small JSON HTTP API. All endpoints are CORS-enabled for origins listed in DJANGO_CORS_ALLOWED_ORIGINS. Free anonymous usage is limited to {{ free_char_limit }} characters/day per IP, {{ max_chars_per_request }} characters/request, and {{ rate_limit }} requests/minute per IP.

GET /api/voices/

Returns the catalog of supported voices. Every voice maps to a real edge-tts / gTTS backend voice — there are no mirrored or fake IDs.

GET /api/voices/?language=ar

Query parameters

NameTypeDescription
languageoptionalISO language code (e.g. ar, en, fr). Omit to list all voices.

Response (200)

{
  "error_code": 0,
  "language": "ar",
  "voices_count": 16,
  "voices_id_list": [1001, 1002, ...],
  "voices_detailed_list": [
    {"id": 1004, "name": "Salma - Egypt Female", "language": "ar",
     "region": "Egypt", "gender": "Female", "engine": "edge",
     "text_characters_limit": 5000}
  ],
  "supported_audio_formats": {{ supported_formats }},
  "ffmpeg_available": true
}

POST /api/tts/

Generate speech audio. Returns a download URL for the rendered file.

POST /api/tts/

JSON body

FieldTypeDefaultDescription
textrequired—Text to synthesize. Max {{ max_chars_per_request }} characters per request.
voice_idrequired—Voice ID from /api/voices/.
audio_formatoptionalmp3One of {{ supported_formats }}.
audio_speedoptional1.00.5 – 2.0 (multiplier).
audio_volumeoptional1.00.5 – 2.0.
audio_pitchoptional1.00.5 – 2.0.
text_paragraph_pause_timeoptional00 = keep newlines, >0 = insert pause, -1 = strip newlines.

Example

curl -X POST http://localhost:8000/api/tts/ \
  -H "Content-Type: application/json" \
  -d '{"text": "مرحبا بالعالم", "voice_id": 1004, "audio_format": "mp3"}'

Response (200)

{
  "error_code": 0,
  "msg": "TTS task executed successfully.",
  "voice": {"id": 1004, "name": "Salma - Egypt Female", ...},
  "audio_download_url": "http://localhost:8000/api/download/tts_1234_abc.mp3",
  "audio_file_format": "mp3",
  "engine_used": "edge",
  "file_size_bytes": 32480,
  "quota": {"daily_limit": 20000, "used": 16, "remaining": 19984, "reset_at": ...}
}

GET /api/quota/

Returns the current daily quota status for the caller's IP.

{
  "ip": "127.0.0.1",
  "daily_limit": {{ free_char_limit }},
  "used": 0,
  "remaining": {{ free_char_limit }},
  "rate_limit_per_minute": {{ rate_limit }},
  "max_chars_per_request": {{ max_chars_per_request }}
}

GET /api/download/<filename>/

Streams a previously generated audio file. Files expire after the configured TTL (AUDIO_FILE_TTL_SECONDS).
